# AKI Launcher

A native Windows launcher for the AKI-engine N64 wrestling game static
recompilations (WCW vs. nWo World Tour, WCW/nWo Revenge, WWF WrestleMania 2000).
Launches each game's recompiled port directly and supports quick-return to the
launcher from a running game.

No ROMs or game assets are included or distributed; each port requires the user's
own ROM.
